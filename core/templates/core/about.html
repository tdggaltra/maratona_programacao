Sobre a Maratona de Programação Brasil - UP

O que é a Maratona de Programação Brasil?

A Maratona de Programação Brasil é um projeto acadêmico que convida os participantes a explorar o Brasil através de desafios de programação. Cada estado brasileiro representa um desafio que deve ser resolvido para avançar até o destino final em Brasília.

Como funciona?

  • Cada estado do Brasil contém um desafio de programação único
  • Os desafios são desbloqueados sequencialmente
  • Ao resolver um desafio, você ganha pontos e desbloqueia o próximo estado
  • Complete todos os desafios dos estados para chegar ao desafio final em Brasília
  • Sua pontuação é baseada no número de desafios completados e na eficiência das suas soluções

Regras

  1. Cada submissão é avaliada automaticamente pelo sistema
  2. As soluções devem ser implementadas na linguagem especificada para cada desafio
  3. As soluções devem respeitar os limites de tempo de execução
  4. Plágio não é permitido e resultará em desqualificação
  5. Você pode enviar múltiplas soluções para cada desafio

Equipe Organizadora

Esta maratona foi desenvolvida como projeto acadêmico pelo Professor Dr. Tiago Dutra Galvão para os estudantes de Engenharia de Software e ADS da Universidade Positivo (Londrina), com o objetivo de promover o aprendizado prático de programação e algoritmos de forma lúdica e interativa.
